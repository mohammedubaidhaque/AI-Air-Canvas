# AI Air Canvas 🎨

## 🚀 Description
This project allows drawing in air using finger gestures with the help of computer vision.

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe

## ✨ Features
- Real-time hand tracking
- Air drawing
- Gesture-based interaction


## 📌 Future Improvements
- Multi-hand detection
- More gestures
